## 2021年秋季学期华中科技大学数据库实验

### 学生管理系统DEMO

功能：

- 新生入学信息增加，学生信息修改。
- 课程信息维护（增加新课程，修改课程信息，删除没有选课的课程信息）。
- 录入学生成绩，修改学生成绩。
- 按系统计学生的平均成绩、最好成绩、最差成绩、优秀率、不及格人数。
- 按系对学生成绩进行排名，同时显示出学生、课程和成绩信息。
- 输入学号，显示该学生的基本信息和选课信息。
